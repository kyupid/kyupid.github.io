# git

## git rm -r --cached .idea/

스테이지에서 파일을 내리고 싶을때 사용한다.
예를 들어, .idea/ 디렉토리와 내부에 있는 모든 파일들을 영구적으로 스테이지에 올리고 싶지 않을때 .gitignore 에 `.idea/` 를 추가하고, 해당 커맨드를 사용할 수 있다.
